## 📝 定制指南

### 1. 基本信息

在 `academic.html` 中修改以下部分：

```html
<h1>Li Leshu</h1>
<p class="title">PhD Candidate / Researcher</p>
<p class="affiliation">Your University</p>
```

### 2. 关于我

更新您的：
- 研究背景和兴趣
- 教育经历
- 研究方向
- 个人兴趣（已包含网球🎾、滑雪⛷️、骑行🚴）

### 3. 研究项目

在 `#research` 部分添加您的研究项目：
- 项目标题
- 时间段
- 项目描述
- 关键词标签

### 4. 发表论文

在 `#publications` 部分添加您的论文：
- 论文标题
- 作者列表（您的名字会用粗体显示）
- 发表会议/期刊
- 论文链接（PDF、代码、项目页面等）

### 5. 教学经历

在 `#teaching` 部分添加您的教学经历。

### 6. 联系方式

更新您的：
- 邮箱地址
- 办公室地址
- 电话号码
- 社交媒体链接（Google Scholar、ResearchGate、ORCID、GitHub等）

### 7. 调整颜色（可选）

在 `academic-style.css` 顶部修改颜色变量：

```css
:root {
    --primary-color: #2c3e50;  /* 主色调 */
    --accent-color: #3498db;   /* 强调色 */
}
```

## 🌐 部署方法

### 方法1: GitHub Pages（推荐）

1. 创建GitHub账号
2. 创建仓库 `username.github.io`
3. 将 `academic.html` 重命名为 `index.html`
4. 上传所有文件到仓库
5. 访问 `https://username.github.io`

### 方法2: Netlify

1. 注册Netlify账号
2. 拖拽文件夹到Netlify
3. 立即部署完成

### 方法3: Vercel

1. 注册Vercel账号
2. 导入项目
3. 一键部署

## 💡 使用提示

1. **头像照片**: 使用正方形、高质量的照片（至少500x500像素）
2. **简历PDF**: 保持文件大小在5MB以下
3. **论文链接**: 确保所有外部链接可访问
4. **定期更新**: 建议每学期更新一次内容
5. **SEO优化**: 在HTML的 `<head>` 中添加meta标签以提高搜索引擎排名
