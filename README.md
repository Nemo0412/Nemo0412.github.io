## Quick Start

1. Open `index.html` in your web browser to view the website
2. Customize the content in `index.html` with your own information
3. Adjust colors and styles in `styles.css` if desired
4. The website is ready to deploy to any web hosting service

## Customization Guide

### 1. Update Personal Information

Open `index.html` and replace the placeholder content with your own:

- **Name**: Search for "Li Leshu" and replace with your name
- **Profile Image**: Replace `IMG_7210.JPG` with your photo filename
- **Resume**: Make sure `LI_LESHU_CV.pdf` is your actual resume file

### 2. About Section

Update the paragraphs in the `#about` section with your personal introduction.

### 3. Experience Section

In the `#experience` section, update:
- Position titles
- Company names
- Dates
- Job descriptions and achievements

### 4. Education Section

In the `#education` section, update:
- Degree names
- University names
- Graduation years
- Academic achievements

### 5. Skills Section

In the `#skills` section, add your own:
- Technical skills
- Tools and technologies
- Soft skills

### 6. Contact Information

In the `#contact` section, update:
- Email address
- Phone number
- Location
- Social media links (LinkedIn, GitHub, Twitter, Instagram)

### 7. Colors (Optional)

To change the color scheme, edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;  /* Main brand color */
    --secondary-color: #1e40af;  /* Secondary brand color */
    --accent-color: #60a5fa;  /* Accent color */
}
```

## Deployment

### GitHub Pages (Free)

1. Create a GitHub account if you don't have one
2. Create a new repository named `username.github.io`
3. Upload all files to the repository
4. Your site will be live at `https://username.github.io`

### Netlify (Free)

1. Create a Netlify account
2. Drag and drop your website folder to Netlify
3. Your site will be deployed instantly with a custom URL

### Vercel (Free)

1. Create a Vercel account
2. Import your project
3. Deploy with one click

## Tips for Best Results

1. **Profile Photo**: Use a high-quality, professional photo (square format works best)
2. **Resume PDF**: Keep your PDF file size under 5MB for faster loading
3. **Content**: Keep descriptions concise and impactful
4. **Contact Form**: The form currently shows an alert. To make it functional, you'll need to integrate with a backend service like:
   - Formspree
   - EmailJS
   - Netlify Forms
   - Your own backend API

## Troubleshooting

**Issue**: Images not showing
- Make sure the image file names match exactly (case-sensitive)
- Ensure images are in the same folder as index.html

**Issue**: PDF download not working
- Verify the PDF file is in the same folder
- Check that the filename matches exactly

**Issue**: Navigation not working on mobile
- Clear your browser cache
- Try opening in a different browser
